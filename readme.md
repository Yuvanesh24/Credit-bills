# Credit Bill — Patient Travel & Others

A simple offline web app to track patient visit travel costs and other bills.

## Features
- Log patients with multiple visits (date, transport, cost, passengers)
- Separate tab for other bills (therapist travel, supplies, etc.)
- Combined grand total across both tabs
- Export per-patient CSV, full CSV, or JSON backup

## Usage
1. **Patients tab** → Add Patient → expand with View ▾ → Add Visit
2. **Others tab** → Add Entry for non-patient bills
3. **↓ CSV** inside a patient panel to download their individual bill
4. **↓ JSON** to back up all data regularly

## Notes
- Data saves in your browser — no backend, no sync across devices
- Clearing browser cache will erase data — keep regular JSON backups
- To move data to another device, export JSON and import on the new one
